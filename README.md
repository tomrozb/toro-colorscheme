## Install
* ln -s ToRo.icls ~/.AndroidStudioPreview/config/colors/
* ln -s ToRo.icls ~/.IdeaIC13/config/colors/


## Set as default
Settings -> Editor -> Colors & Font -> Choose "ToRo" -> OK
